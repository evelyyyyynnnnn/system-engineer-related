# BrewMeet - 咖啡聊天应用

一个类似Hinge风格的咖啡聊天应用，专注于帮助用户找到合适的咖啡聊天伙伴，而不是约会对象。

## 功能特性

### 🎯 核心功能
- **发现咖啡伙伴**: 浏览和匹配潜在的咖啡聊天伙伴
- **个人资料**: 展示个人信息、咖啡偏好、兴趣爱好等
- **匹配管理**: 查看和管理你的匹配和对话
- **精选推荐**: 基于算法推荐的高质量匹配用户
- **咖啡邀请**: 接收和发送咖啡聊天邀请
- **设置中心**: 个性化设置和偏好管理

### 📱 页面结构
1. **DiscoverPage** - 发现页面，类似Hinge的滑动卡片界面
2. **StandoutsPage** - 精选推荐页面，展示算法推荐的高质量用户
3. **InvitationsPage** - 咖啡邀请页面，管理收到的邀请
4. **MatchesPage** - 匹配页面，查看所有匹配和对话
5. **ProfilePage** - 个人资料页面，展示和管理个人信息

### 🎨 设计特色
- **简约风格**: 参考Hinge的简洁设计语言
- **咖啡主题**: 专注于咖啡聊天而非约会
- **中文界面**: 完全中文化的用户界面
- **现代UI**: 使用SwiftUI构建的现代化界面
- **黑白配色**: 采用经典的黑白配色方案，简洁优雅

## 技术实现

### 开发环境
- **平台**: iOS 26.0+
- **语言**: Swift
- **框架**: SwiftUI
- **最低支持**: iOS 26.0

### 项目结构
```
BrewMeet/
├── BrewMeetApp.swift          # 应用入口
├── ContentView.swift          # 主界面和底部导航
├── Models.swift               # 数据模型
├── Components.swift           # 可复用UI组件
├── DiscoverPageView.swift     # 发现页面
├── ProfilePageView.swift      # 个人资料页面
├── MatchesPageView.swift      # 匹配页面
├── StandoutsPageView.swift    # 精选推荐页面
├── InvitationsPageView.swift  # 咖啡邀请页面
└── SettingsPageView.swift     # 设置页面
```

### 主要组件
- **Card**: 通用卡片组件
- **CustomButton**: 自定义按钮组件
- **Badge**: 标签组件
- **ProfileCard**: 用户资料卡片
- **AsyncImageWithFallback**: 带占位符的异步图片组件

## 如何运行

### 在XCode中运行
1. 打开 `BrewMeet.xcodeproj` 文件
2. 选择目标设备（iPhone模拟器或真机）
3. 点击运行按钮 (⌘+R)

### 支持的设备
- iPhone 模拟器
- 真机设备 (iOS 26.0+)

## 功能说明

### 发现咖啡伙伴
- 滑动浏览用户卡片
- 点击"约咖啡"按钮表示兴趣
- 点击"X"按钮跳过用户
- 支持重新开始浏览

### 个人资料
- 展示个人信息和照片
- 咖啡偏好设置
- 兴趣爱好标签
- 空闲时间设置
- 常去咖啡店列表
- 聊天话题标签

### 匹配管理
- 查看新匹配
- 管理对话列表
- 状态标签（新匹配、聊天中、计划中、已约定）
- 未读消息提醒

### 精选推荐
- 基于算法的智能推荐
- 推荐理由展示
- 高质量用户筛选

### 咖啡邀请
- 接收其他用户的邀请
- 查看邀请详情（地点、时间、咖啡类型）
- 接受、拒绝或聊天回复

### 设置中心
- 发现偏好设置（年龄范围、距离、咖啡偏好）
- 隐私与安全设置
- 通知管理
- 空闲时间设置
- 帮助与支持

## 数据模型

### 用户模型 (User)
- 基本信息：姓名、年龄、照片
- 个人描述：简介、职业、位置
- 咖啡偏好：喜欢的咖啡类型
- 兴趣爱好：标签列表
- 时间设置：空闲时间、常去地点

### 匹配模型 (Match)
- 匹配信息：用户信息、匹配时间
- 对话状态：最后消息、时间戳
- 状态管理：新匹配、聊天中、计划中、已约定

### 邀请模型 (Invitation)
- 邀请详情：邀请者信息、邀请消息
- 建议信息：地点、时间、咖啡类型
- 共同兴趣：标签列表

## 设计理念

这个应用专注于**咖啡聊天**而非约会，旨在：
- 帮助用户找到有共同兴趣的聊天伙伴
- 促进有意义的面对面交流
- 营造轻松友好的社交氛围
- 专注于咖啡文化和深度对话
